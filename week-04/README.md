#Capstone 1

In this project, I analyze Bacardi’s performance using the Iowa Liquor Sales database. This capstone highlights my ability to use SQL for filtering, sorting, aggregating, joins, and subqueries. 
I also exported my query results into Excel to build pivot tables and visualizations that support a full SWOT analysis for the brand.
